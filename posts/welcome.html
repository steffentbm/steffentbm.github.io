Hello world! Welcome to my new corner of the internet.

I decided to build this blog from scratch using only the fundamental building blocks of the web: HTML, CSS, and JavaScript. No heavy frameworks, no complex build tools, just simple, clean code.

Why Minimalist?

In a world of information overload, I wanted a space that feels calm and focused. The design reflects this philosophy:

  • High Contrast: Easy to read text.
  • Clean Typography: Using Inter for a modern look.
  • Fast Performance: No bloat means instant loading times.

Stay tuned for more updates as I continue to tweak and improve this site!
